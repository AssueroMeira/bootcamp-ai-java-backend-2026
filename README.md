# Bootcamp AI Java Back-end 2026

Este repositório reúne minha evolução, anotações, exercícios, desafios e projetos desenvolvidos durante o **Bootcamp AI Java Back-end 2026**.

## Objetivo

Documentar minha jornada de aprendizagem em desenvolvimento Java Back-end, com foco em:

- fundamentos da linguagem Java;
- orientação a objetos;
- boas práticas de desenvolvimento;
- Maven e Gradle;
- persistência de dados com JDBC;
- Spring Boot;
- APIs REST;
- segurança com Spring Security;
- consumo de APIs externas;
- aplicações com inteligência artificial no back-end.

## Estrutura do repositório

### 00 — Ambiente e Ferramentas

Configuração inicial do ambiente de desenvolvimento.

- SDKMAN
- Amazon Corretto JDK
- WSL
- VS Code
- Git e GitHub

### 01 — Copilotos com IA no Desenvolvimento de Soluções

Módulo introdutório sobre uso de IA, copilotos, LLMs, agentes e engenharia de prompts.

### 02 — Fundamentos para Começar a Programar em Java

Primeiros contatos com Java, sintaxe, lógica de programação, estruturas de controle e desafios iniciais.

### 03 — Programando o Mundo Real com Orientação a Objetos no Java

Estudo de classes, objetos, encapsulamento, herança, polimorfismo, interfaces, lambda, collections e exceções.

### 04 — Boas Práticas, Padrões e Dados no Java

Estudo de Maven, Gradle, annotations, SOLID, Clean Code, JDBC, persistência de dados e padrões de projeto.

### 05 — Potencializando sua Produtividade com Spring Boot

Criação de APIs REST com Spring Boot, conexão com banco de dados, Spring Data, Spring Security e OpenFeign.

### 06 — Projeto Final: Assistente Virtual Inteligente por Voz

Projeto final envolvendo API inteligente, reconhecimento de fala e Spring Boot.
