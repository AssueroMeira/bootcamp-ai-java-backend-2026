# Fundamentos da Sintaxe da Linguagem Java

Esta pasta reúne anotações e exemplos práticos sobre os primeiros conceitos da sintaxe Java.

## Conteúdos estudados

- Estrutura básica de uma classe Java
- Método `main`
- Variáveis
- Tipos primitivos
- Comentários no código
- Comentários de documentação
- Criação e chamada de métodos
- Escopo de variáveis
- Entrada de dados com `Scanner`
- Declaração explícita de variáveis
- Inferência de tipo com `var`

## Objetivo

Registrar exemplos simples e comentados para facilitar a revisão futura e demonstrar minha evolução no aprendizado de Java Back-end.

## Como executar os exemplos

Entre na pasta `exemplos`:

```bash
cd exemplos
```
